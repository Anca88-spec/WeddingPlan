# Wedding Planner - Console Project

This is a Java project for wedding planning, implemented as a console application.

## Description

The project allows managing guests, budgets, and vendors for a wedding.  
It was created for educational purposes and to demonstrate Java programming skills.

## Features

- Manage guests (add, remove, list)
- Manage expenses and budget
- Manage vendors
- Create and view the wedding plan

## How to Run

1. Open the project in IntelliJ IDEA or another Maven-compatible IDE.
2. Run the `ProjectApplication.java` class.
3. Follow the console instructions to test the features.
